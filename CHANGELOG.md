# Changelog

## 0.3

- Přidán čistý protokol `APP_*` mezi Arduinem a Windows aplikací.
- Přidán viditelný surový log komunikace Arduino / ECU.
- Přidán RX monitor v aplikaci.
- Přidáno automatické hledání COM portu.
- Přidáno čtení paměti závad ECU přes KW1281.
- Přidáno mazání paměti závad s potvrzením.
- Přidána živá data: otáčky, napětí baterie, úhel škrticí klapky.
- Přidána základní česká databáze běžných VAG/OBD závad.
- Ověřeno na ECU `047906030N`, `SIMOS 2P`, KW1281 adresa `0x01`.
